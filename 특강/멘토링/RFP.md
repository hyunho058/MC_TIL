# RFP

> Request for Proposal

* 제안 요청서
* 프로젝트 의뢰서
* 사업 발주 의뢰
* 기타

## 프로세스

1. 기획
2. 제안 요청서 작성
3. 요청서 심사
4. 제안서 작성
5. 제안서 심사
6. 프로젝트 시작 및 시작
7. 프로젝트 발표

## RFP 주요 사항

* 목표 설정
* 요구사항 명세
* **구현 요소 정리**
  * ex ) 무게를 감지하는 조건..
* 제약 조건 도출
* 일정 수립(구현 가능한 일정)

### 지향 요소

1. 명확성(단순성)
2. 가능성
3. 자원 적극 활용
